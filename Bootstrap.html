Bootstrap

Bootstrap is a popular open-source front-end framework for building mobile-first and responsive websites. Bootstrap, developed by Twitter, is a collection of HTML, CSS, and JavaScript components, that can help developers build websites faster and more effectively. With Bootstrap, developers can properly align content across all device sizes (desktop, tablet, and mobile) using the grid system with 12 columns. The elements that come built-in (e.g., buttons, forms, navigation bars, modals, cards, etc.) are all styled responsively and consistently. To use Bootstrap in an HTML document, you need to include the CSS and JavaScript files for Bootstrap, which can be either downloaded onto your server or from a CDN. When you add Bootstrap to your document, you'll be ready to apply styles to your HTML tags and layout, all using Bootstrap classes.

Common Bootstrap Classes

Bootstrap Class / Element Purpose / Description Syntax Example
.container Wraps and centers content with padding <div class="container">Content</div>
.row, .col-* Grid layout system (rows and columns) <div class="row"><div class="col-6">Half</div></div>
.btn, .btn-primary Styled buttons <button class="btn btn-primary">Click</button>
.form-control Styled form inputs <input class="form-control" type="text">
.navbar, .navbar-brand Navigation bar structure <nav class="navbar"><a class="navbar-brand">Brand</a></nav>
.card, .card-body Card layout for content blocks <div class="card"><div class="card-body">Text</div></div>
.text-center Centers text horizontally <p class="text-center">Centered Text</p>
.img-fluid Makes images responsive <img src="image.jpg" class="img-fluid">
.alert, .alert-success Display alert messages <div class="alert alert-success">Success!</div>
.table, .table-striped Styled tables with optional striped rows <table class="table table-striped">...</table>
.badge, .bg-danger Small badge indicators <span class="badge bg-danger">New</span>
.text-danger, .text-success Text color utilities <p class="text-danger">Error</p>
.d-flex, .justify-content-center Flex layout utilities <div class="d-flex justify-content-center">Centered</div>
