Cascading Style Sheets

CSS (Cascading Style Sheets) is a language for styling and marking up HTML web pages. While HTML provides a web page's structure, CSS manages how that structure is visually disposed (fonts, colors, spacing and placement). Having CSS separate content from design makes web development more orderly and efficient. CSS can apply styles to an individual element, groups of elements, or entire pages.
