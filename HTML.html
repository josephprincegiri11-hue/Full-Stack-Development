Hyper Text Markup Language

Web pages are frequently created and programmed using HTML (Hyper Text Markup Language). It uses a system of elements and tags to organize web content. HTML instructs the browser on how to present links, text, images, and other types of material. Tags are used to represent the elements that make up HTML documents. Angle brackets encompass keywords, such as <p> for paragraph. The content is positioned between the opening and closing tags of the majority of HTML elements (e.g.<p>..</p>) An HTML document is structured like a tree, <!DOCTYPE html> with the declaration at the top and the root element below it. The two main parts of <html> are<head> and <body>. All the visible information, such as headings, paragraphs, links, and images, is located in the <body>, but meta information, such as the page title, is located in the <head>.

Common HTML Tags

Tag Purpose Example
<html> Root of an HTML document <html>...</html>
<head> Contains metadata <head>...</head>
<title> Sets the page title <title>My Page</title>
<body> Main content of the page <body>...</body>
<h1> Main heading <h1>Welcome!</h1>
<p> Paragraph <p>This is a paragraph.</p>
<a> Hyperlink <a href="https://example.com">Visit</a>
<img> Image <img src="image.jpg" alt="Image">
<ul> Unordered list <ul>...</ul>
<li> List item <li>Item 1</li>
<div> Block-level container <div>Content</div>
<span> Inline container <span>Text</span>
<br> Line break Line 1<br>Line 2
<input> Form input field <input type="text">
<form> Form element <form>...</form>
